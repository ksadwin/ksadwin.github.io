SKKS Stylometry Engine

Sam Kamenetz and K Sadwin's Authorship Identification and Stylometry system developed for CS490 - Search Engines and Recommender Systems.

Using natural language processing and the Project Gutenberg Library, the SKKS was an attempt to cluster authors by writing style according to vocabulary size, sentence length, and percentage of adjectives and adverbs.

  • Date: Spring 2015
  • Platform: Python 3.4

Python Search Engine

A Python-based search engine built through Doug Turnbull's Search Engines and Recommender Systems class at Ithaca College. Developed by K Sadwin and Jen Westling, using examples provided in class.

  • Date: Spring 2015
  • Platform: Python 3.4

The Dating Language

In-progress senior project with Beth Dellea. The goal is to scrape OkCupid profiles for user demographics and profile contents (complete!) and use natural language processing to find differences in language use between users of different genders, orientations, etc.

  • Date: Fall 2015 (ongoing)
  • Platform: Python 3.4
